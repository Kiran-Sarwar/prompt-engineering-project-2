# Prompt Engineering Project 2: Chain-of-Thought (CoT) Logic Engine

## Overview

This project was completed as part of the DecodeLabs Prompt Engineering Internship.

The objective is to design a prompt that improves an AI model's reasoning by encouraging structured thinking, self-correction, and accurate responses to logic-based problems.

## Objectives

- Build a Chain-of-Thought (CoT) prompt.
- Reduce hallucinations through structured reasoning.
- Include a self-correction phase.
- Test the prompt using logic and reasoning challenges.
